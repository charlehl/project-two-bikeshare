Metro Bike Share


According to the National Association of City Transportation Officials (NACTO), since 2010 there has been 123 millions bike trips taken on bike share. In 2017 alone, 35 millions trips were taken which is a 25% increase compared to the previous year of 2016. For more statistical information, please visit NACTO website.

Some of the benefits of Bike Share are:
- Cycling health benefits for the individual;
- Co2 emission reduction;
- Local traffic reduction;
- Increase in mobility which benefits businesses.


To accomplish this we used the following:
- Python Flask;
- JavaScript;
- HTML;
- CSS;
- MongoDB;
- GeoJson file;
- Mapbox API;
- New libraries: jquery and ajax;
- Data source: LA Metro Bike Share.


This Bike Share project focuses on the data retrieved from the Los Angeles Metro Bike Share to build a full stack website with:
1) A map with all the stations, providing the user cycling directions from station to station, and driving directions from any location to a selected station;
2) A dashboard where the user can select the station name and get the live station status and at the same time, visualize that station peak hours;
3) Charts displaying data analysis on Pass Type.
