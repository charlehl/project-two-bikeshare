Bike Share in L.A.


Project Description(Outline)
Explore the bike share trend in Los Angeles. Try to classify bike share users by analyzing rental period, travel distance, subscription type, usage by season and so on.

Data Sets to be Used
Please click here to see our data source. On this website we will use their JSON files on the stations status, csv files on trip data and station information.

Proposed layout
Build a full stack website hosting:
- An interactive map, showing a navigation from start station to end station
- A dashboard displaying information on stations status and bikes availability
- Interactive D3/plotly visualizations

Tools
- MongoDB
- Flask
- HTML, CSS
- JavaScript (JQuery, leaflet, Mapbox)

Inspirations

Interactive map with live navigation showing the way from start station to end station.

Interactive visualizations with multiple x-axis and y-axis, not necessarily bubble plot, we still need to explore our dataset to see what data would be interesting to plot.

Create a dashboard similar to this one to display bike stations status, address, bike availability.
